本文件夹存放的是，2026NGC都市天际线拜年纪，第十二号作品——《岩庆》的相关资源。  
内容包括：  
- 存档本体  
- 订阅的模组与资产列表  
- 游戏与模组的所有设置  
- 游戏运行环境  
- 剪辑工程  

# 存档本体
请从 releases 页面下载，或前往[Paradox Mods](https://mods.paradoxplaza.com/mods/135370/Any)下载
